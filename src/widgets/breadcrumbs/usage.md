# Breadcrumbs

Breadcrumbs is a widget that provides breadcrumb styling for its link children

## API

### Breadcrumbs

```jsx
import { Breadcrumbs, Link } from "react-simple-widgets";

<Breadcrumbs>
    <Link to="">Home</Link>
    <Link to="">Page 1</Link>
    <Link to="">Path 1</Link>
    <Link to="">Sub-path 1</Link>
    <Link to="">Sub-sub-path 1</Link>
</Breadcrumbs>
```
